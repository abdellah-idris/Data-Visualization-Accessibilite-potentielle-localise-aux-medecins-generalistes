# DataVis


Interactive France Map about "Accessibilité potentielle localisée aux médecins généralistes"

## Table of Contents

- [Installation](#installation)
- [Usage](#usage)

## Installation

Clone project
#### Active venv
```bash
.\venv\Script\activate
```

#### Install requirments
```bash
pip install -r requirements.txt
```

## Usage


#### Launch map
```bash
python multimap.py
```

#### Launch Chart
```bash
cd age_distribution
```


```bash
python -m http.server
```
